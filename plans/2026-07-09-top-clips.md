# Top Clips — evidence-grounded clip recommendations

## Goal

Scan the RECALL library and surface ranked clip candidates with three provenance
labels: **PROOF** (the line matches a hook from the user's HOOKLAB ledger with
outcome=winner, or strongly contains a high-evidence pattern scaffold),
**AI + PROOF** (the AI flags the line as a close variation of a proven
pattern/winner, with a grounding note naming it), **AI RECOMMENDED** (pure AI
judgment, honestly labeled). Works with no API key (PROOF only, fully offline);
a key unlocks the AI pass. Channel name + niche in Settings personalize ranking.
Principles: provenance on every candidate; the AI never invents the pattern set;
nothing is ever rendered as a virality score.

## Steps

1. `topclips-patterns.js` — vendored snapshot of HOOKLAB's 97-pattern bank
   (reduced fields), used as fallback; at runtime Top Clips upgrades to a live
   dynamic import of `../Hooklabs/patterns.js` when the sibling app is reachable
   on the same origin (as it is on the github.io deploy).
2. `topclips.js` (`window.TopClips`) — offline matcher (ledger jaccard ≥ 0.55;
   scaffold containment ≥ 0.75 for skeletons ≥ 4 tokens, exact for 3-token
   skeletons; only patterns with strength ≥ 0.8 and evidence ≠ hypothesis;
   4–40-word gates; chunked 400 segments/frame), ranked by
   evidence·similarity + specificity + niche bonus (sort keys only, never shown);
   mode-render into `#results` reusing the `.res` card + `+ BIN` wiring; ledger
   read directly from `hooklab_state_v1` (shared origin) with a graceful hint
   when absent.
3. AI pass (commit 2) — one `LLMProvider.generateText` jsonMode call; candidates
   by compact index; offline PROOF immutable; `ai_proof` demoted to `ai` unless
   its grounding names a real pattern or ledger winner; errors leave offline
   results intact. CHANNEL settings block (name + niche) stored as additive keys
   in `recall_settings_v1`.
4. `index.html` TOP CLIPS button + script order; `style.css` gold label/badge
   styles; `sw.js` cache v10→v11, APP_SHELL += topclips files **and llm.js
   (fixes a pre-existing omission)**.
