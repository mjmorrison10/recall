How to use RECALL

Three things: how to use it, how to install it on your phone, and how to get a Gemini API key for audio transcription.

1. Using RECALL 2. Installing RECALL 3. Getting a Gemini API key
1

Using RECALL

RECALL is searchable memory for everything you watch and listen to. Drop in transcripts, search a phrase, get every moment across every source. Stack the moments, copy the shot list, cut.

01Add a source

Click + add source. Give it a title (e.g. Tate x Fresh & Fit — Ep 214). Then either:

  • Paste a transcript — any timestamped transcript. One line per moment, each starting with a timecode like [00:01:23], 1:23, or 01:23:45.
  • Upload audio or video — drag in an .mp3, .m4a, .wav, or video file. RECALL sends it to Google Gemini for transcription (requires an API key — see Section 3).

02Search

Type a word or phrase in the search box. RECALL surfaces every moment it appears in across every source you've loaded — timecoded, highlighted, ready to grab.

03Toggle sources

Click any source chip at the top to turn it on or off. Search results narrow to the sources you've got switched on. Useful when you've got 12 podcasts loaded and only want to search 2.

04Bin moments

Hit + BIN on any moment you want to stack. They pile up in the Clip Bin on the right. The best clips aren't one clean section — they're a line here, a callback there. This is where that gets built.

05Export

Hit EXPORT. RECALL copies a clean shot list — numbered, timecoded, with source labels — to your clipboard. Paste it into your notes app, take it into your editor, go cut.

Tip: transcripts and your bin live in your browser's local storage. Nothing leaves your device. To back up your whole library — including all sources and your current bin — open ⚙ SETTINGS → DATA → Export library. That saves a JSON file you can re-import on another device or after clearing browser data.

Installing RECALL

RECALL is a Progressive Web App. You can install it to your phone or desktop so it launches like a native app — no App Store, no Google Play, no download. Your library stays on your device and the app works offline once installed.

iPhone · Safari

Add to Home Screen

Open https://mjmorrison10.github.io/recall/ in Safari.

  1. Tap the Share button (square with up arrow, bottom of screen).
  2. Scroll the action sheet and tap Add to Home Screen.
  3. Confirm the name "RECALL" and tap Add.

RECALL appears on your home screen as a cyan app icon. Tap it → launches fullscreen, no Safari chrome.

If you already have an old RECALL shortcut: long-press it → Remove App, then re-add. iOS caches the icon from the first install and won't refresh it until you remove and re-add.
Android · Chrome

Install app

Open https://mjmorrison10.github.io/recall/ in Chrome.

  1. Chrome will prompt you to install automatically — tap Install. Or tap the menu (⋮) → Install app.
  2. RECALL appears in your app drawer with the cyan aperture icon.
  3. Tap it → launches fullscreen, no Chrome chrome.
Desktop · Chrome or Edge

Install as window app

Open https://mjmorrison10.github.io/recall/ in Chrome or Edge.

  1. Click the install icon (⊕) in the right side of the URL bar.
  2. Click Install.

RECALL opens as a standalone window with its own taskbar entry — like a regular desktop app.

Getting a Gemini API key

You only need a key if you want RECALL to transcribe audio or video files. If you only paste existing transcripts, skip this whole section.

01Open Google AI Studio

Go to aistudio.google.com/app/apikey in any browser. Sign in with your Google account if you're not already.

02Create a key

Click Create API key. If you don't have a project yet, Google will create one for you and the key is generated instantly. The key is a long string starting with AIza.

03Copy the key

Click the copy icon next to your new key. Don't try to type it out — these are 39 characters of random text.

04Paste it into RECALL

Open RECALL. Click ⚙ SETTINGS in the top bar. Paste your key into the field and click Save. RECALL stores it in your browser's local storage and sends it to Google Gemini only when transcribing audio.

05Test it

Click + add source, drag in a 1–2 minute audio clip, hit Add to library. If you get a green toast saying "247 moments added", you're set. If you get "API key rejected", the key has a typo — clear it in Settings and paste again carefully.

Cost: Gemini 1.5 Flash is roughly $0.009 per hour of audio transcribed. The free tier covers most personal use without paying anything.
Privacy: Your key is stored only in your browser. RECALL sends it to Google only when calling Gemini for transcription. Don't paste your key anywhere public — if it leaks, go back to aistudio.google.com and revoke it, then create a new one.
